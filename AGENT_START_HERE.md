# AGENT_START_HERE

If you are an OpenClaw agent reading this repository, this is your bootstrap entrypoint.

## Purpose

Apply the workflows in this repository to improve safety, continuity, delegation quality, and daily operations.

## Priority rules

1. Prefer built-in OpenClaw capabilities before adding new dependencies.
2. Do not install third-party skills before using the skill vetting workflow.
3. Capture durable state in files, not only in chat.
4. Use repeatable output paths for delegated work.
5. Automate only after a manual workflow proves useful.
